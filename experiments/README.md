# Experiments

This directory stores loop-scoped experiment folders.

Recommended structure:

```text
experiments/
  <loop_id>/
    loop_plan.md
    config_snapshot.json
    run_summary.json
    report.md
    regression_cases/
    winning_cases/
```

Use one `loop_id` per focused iteration. Keep product-track and research-track work in separate loop folders when possible.

The `loop_template/` folder provides a ready-to-copy scaffold.
